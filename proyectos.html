Sistema Web Multitenant - TAUP Agency

Trabajé como principal diseñador y desarrollador de la UI/UX de este sistema de gestión de empresas multitenant (multiempresa). El mismo fue desarollado con Next.js y TypeScript, utilizando la librería de Shadcn/ui en conjunto con tailwind.css y usé herramientas como Zustand para la gestión de estado global, Zod para validación de formularios. No puedo mostrar el contenido real de la app porque es de uso privado pero poseo una demo acá. Ingresá con "admin" y "admin123"!

Trabajo como Freelance

Proyecto Freelance, desarrollado con Next.js, TypeScript y la librería de componentes Shadcn/ui en el front y Spring Boot, JWT y Supabase (PostgreSQL) para el back. El proyecto fue encargado por un tatuador independiente de Rosario que necesitaba un sistema para gestionar los turnos de su estudio y además permitir que sus clientes puedan sacar turno en el momento que quisieran y cuando ellos decidan. Pruebas Técnicas

Tablero KanBan con Web Sockets - Next.js, Nest.js y MongoDB + N8N

Proyecto Full-Stack realizado con Next.js, Nest.js y MongoDB para una prueba técnica Full-Stack. Se solicitó recrear un tablero KanBan con columnas personalizables, notas editables y movibles con drag & drop fluido y web sockets para (usé Socket.io) para edición en tiempo real. Además, use N8N para automatizar la descarga y envío a un mail de elección los datos actuales del tablero; los mismos se envían en formato CSV. Es de acceso público, asi que podés organizar tu día y luego exportarlo acá.

Food-Truck Finder - Next.js, API Routes y Leaflet

Proyecto Full-Stack realizado con Next.js para una prueba técnica. La idea principal es que un usario (localizado en San Francisco, US) pueda encontrar Food-Trucks en base a ciertos parámetros de búsqueda como distancia o tipo de comida. Desarrollé un pequeño Back-End gracias al API Routes de Next.js y usé Tailwind para estilizar. Además, use la librería de código abierto leaflet para el mapa. Podes revisarlo acá.

Menu para Restaurante - React.js, Tailwind y Material UI

Menu interactivo hecho en React.js para una prueba técnica. Simula una carta de comidas que permite operaciones CRUD sobre un endpoint simulado a través de métodos HTTP. Usé React.js, Tailwind y la librería de componentes Material UI y usé Vercel para el deploy. Además, use MockAPI para simular la BDD. Podés interacturar con la carta directamente desde acá.
