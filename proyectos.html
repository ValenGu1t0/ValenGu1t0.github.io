Mis Páginas Web

En esta sección podes ver y revisar el código las primeras páginas web que hice (pero no las últimas). Como me dedico plenamente al front end, podes revisar que tecnologías utilice en la sección de tecnologías.

Mi primer Landing Page

Después de haber pasado por incontables cursos de Youtube, decidi que necesitaba desafíos que me mantengan codeando activo, por lo que decidí buscar un curso online, que sea dinámico y con objetivos. Así descubrí FreeCodeCamp, al que le atribuyo gran parte de mis conocimientos. Cuando terminé el curso de Responsive Web Design, sentia que estaba preparado para hacer mi primer "good-looking" web, asi que me decidí a probar y este fue el resultado.

Página para una Empresa de Construcción

Un dia mi papa me comento que necesitaba una página web, pues habia contratado uno de esos servicios que hacen web a mansalva y de forma industrial, sumado a que no podia casi modificar cosas y ni hablar de que no lo contactaba nadie. Como yo estaba aprendiendo desarrollo web pense que sería una buena idea si me encargaba yo de ahora en adelante, asi que eso hice, y este fue el resultado. Sin embargo, el proyecto nunca termino de concretarse pues tenia que dedicarme a mi carrera, asi que deje el proyecto en stand by. Quiza algún dia vuelva a retocarla o simplemente haga una nueva.

Videojuego RPG en Javascript (El sueño del pibe)

Después de pasar mucho tiempo aprendiendo maquetar bien y hacer páginas responsive con un diseño atractivo, sentia que era hora de empezar con Javascript. A pesar de haber programado en C, C++, Java e incluso Haskell, sentía que debía amigarme con la lógica de Javascript para dominar el Front End. Así que decidi seguir aprendiendo y empecé el curso de JavaScript Algorithms and Data Structures de FreeCodeCamp. Uno de los primeros grandes desafíos fue hacer un juego chico de RPG, con toma de decisiones e incluso mejoras de personaje (lo cual me encanta pues soy un aficionado de los videojuegos). Podes echarte una partida a ver si te gusta lo que programé con ayuda de FreeCodeCamp.

To Do App con Javascript

A medida que fui aprendiendo Javascript y sus eventos, decidi poner en práctica lo aprendido con una simple app donde podes agregar y eliminar tareas. No es muy compleja en cuanto a la lógica pero me sirvió mucho para jugar con el DOM y los eventos en JS. Podes revisarla acá y anotar esas cosas que venis pateando hace rato para cumplirlas de una vez por todas!

Palindrome Checker con Javascript

Para hacer el trabajo final de la primer sección de FreeCodeCamp hice una página que chequea por si un string es palíndromo o no. De paso me di el lujo de hacer la página en Tailwind CSS y quedé satisfecho con su simplicidad.
